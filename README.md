# Todo List

This is a simple Todo List web application built using HTML, CSS, and JavaScript. I created this project to improve my understanding of JavaScript, DOM manipulation, and browser Local Storage.

The application lets users add tasks, mark them as completed, and delete them. All tasks are stored in the browser, so they remain available even after refreshing the page.

## Features

- Add new tasks
- Mark tasks as completed
- Delete tasks with a right click
- Tasks are automatically saved using Local Storage
- Simple and clean user interface

## Technologies Used

- HTML
- CSS
- JavaScript
- Local Storage API

## How to Run

1. Clone this repository.

```bash
git clone https://github.com/kowshik459/Todo-List.git
```

2. Open the project folder.

3. Open `index.html` in your browser.

That's it! No additional setup is required.
